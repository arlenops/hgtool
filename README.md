# HGTool - 黑果云现代运维工具箱

> 基于 Bash 开发的模块化运维工具集，零外部依赖，赋予您极简、高效、自动化的服务器管理体验。

## 🚀 核心特性

- **开箱即用**: 纯 Bash 编写，无需安装 Python/Node 等外部运行环境，一条命令即可启动。
- **模块化架构**: 功能高度解耦，基于插件目录结构设计，轻松进行二次开发和功能扩展。
- **优雅终端 UI**: 内置高级 ANSI UI 渲染引擎，支持方向键交互菜单、色彩丰富的提示与进度动画。
- **系统适配**: 自动适配主流 Linux 发行版（CentOS / Ubuntu / Debian）。

## 🧰 工具箱功能矩阵

- **系统自检**: 硬件、网络及系统健康度全面检测与报告。
- **软件源管理**: 一键更替国内镜像源（中科大、清华、阿里云、腾讯云等）及恢复官方源。
- **存储管理**: 硬盘格式化自动挂载、分区扩容、大文件与磁盘占用深入分析。
- **系统管理**: 系统安全更新、时区一键设置、Swap 虚拟内存动态配置。
- **网络与 DNS 管理**: DNS 切换与联通性测试、修改 SSH 端口、防火墙配置、邮件端口(25)检测。
- **SSH 安全管理**: 极速配置允许/禁止 Root 登录、密码认证、密钥认证。
- **Web 与容器化运维**: 
  - **[宝塔面板]** 一键安装/卸载、找回/强置密码、修改访问端口、全自动挂载数据盘及数据迁移。
  - **[Caddy]** 快速部署最新版、完全卸载与守护进程状态查看。
  - **[Docker]** 官方/国内加速镜像安装、废弃与悬空镜像一键清理。

## 🛠️ 快速安装与使用

请在您的 Linux 服务器上执行以下命令：

```bash
git clone https://github.com/arlenops/hgtool.git
cd hgtool
sudo bash hgtool.sh
```

*注：建议使用 `root` 权限（或通过 sudo）运行，以确保修改源、挂载硬盘及安装服务等底层权限能正常作用。*
